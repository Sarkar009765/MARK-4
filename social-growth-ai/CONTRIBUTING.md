# Contributing to Social Growth AI

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/MARK-4.git`
3. **Create** a new branch: `git checkout -b feature/your-feature`
4. **Make** your changes
5. **Test** your changes: `python main.py status`
6. **Commit** your changes: `git commit -m "Add feature"`
7. **Push** to your fork: `git push origin main`
8. **Create** a Pull Request

## Code Style

- Follow PEP 8 for Python code
- Use meaningful variable names
- Add docstrings to functions
- Keep functions small and focused

## Testing

Before submitting, test your changes:
```bash
python main.py status
python main.py full
```

## Issues

- Report bugs: https://github.com/Sarkar009765/MARK-4/issues
- Feature requests welcome!
